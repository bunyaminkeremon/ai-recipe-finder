# AI Recipe Finder

An AI-powered recipe finder that suggests recipes based on the ingredients you have. Add your ingredients, pick a cuisine and diet preference, and get 3 detailed recipes with ingredients and step-by-step instructions.

## Live Demo

[Try it here](https://bunyaminkeremon.github.io/ai-recipe-finder/)

## What It Does

- Add ingredients one by one with the + button
- Remove ingredients by clicking x
- Optional cuisine and diet filters
- AI generates 3 recipes with cooking time, difficulty, ingredients and instructions
- Powered by Google Gemini API

## Tech Stack

- HTML5
- CSS3
- JavaScript (async/await, Fetch API)
- Google Gemini API

## Run Locally
```bash
git clone https://github.com/bunyaminkeremon/ai-recipe-finder.git
```

Open `index.html` in your browser. You'll need your own API key from aistudio.google.com — replace the key in `script.js`.
